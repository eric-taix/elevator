Elevator
========

Elevator code story implementation in Dart. Full rules of this code story can be found [here](http://www.code-story.net/blog/posts/s03e01)


Build status
============

[![Build Status](https://drone.io/github.com/eric-taix/elevator/status.png)](https://drone.io/github.com/eric-taix/elevator/latest)

This project use [Drone.io](https://drone.io) as a continuous build platform. When build is successful, it is automatically deployed to [Heroku](https://www.heroku.com).

The URL of this application is [http://elevator-taix.herokuapp.com/](http://elevator-taix.herokuapp.com/).
